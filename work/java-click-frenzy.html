Click Frenzy
A 2D block clicking game with upgrades and abilities
JavaFX
Background
A 2D blocky game with upgrades
This project is a step towards learning object oriented programming.
The aim of this game is to get the highest score in a set amount of time. Blocks are generated at random: green, blue, pink and gold blocks. Each colour represents the rarity, each carrying a set number of points.
Upon collecting a number of points, they can be used to upgrade skills, such as score multiplier and timer increments.
Blocks
Block scores and variety
Each block has a score that can be collected. Click as many higher quality blocks to get the most score.
Common
Green block
Each green adds +1 to the score.
Rare
Blue block
Each blue adds +2 to the score.
Epic
Purple block
Each purple adds +3 to the score.
Legendary
Gold block
Each gold adds +5 to the score.
Trap
Grey block
Each grey removes -5 from the score.
Upgrades
Claim upgrades and push your score further
Upon collecting a set number of points, an upgrade can be purchased.
Score Multiplier
Increase your score by a multiplier
Six stages (1, 2, 5, 10, 20, 50) to get, with each unlocked after the previous is attained.
Time Extender
Extend the time during the gameplay
Six stages (60, 70, 80, 90, 100, 100) in seconds to get.
Abilities
Randomised abilities help boost the score even more
Upon clicking a set number of times, an ability will be granted randomly.
Doubled Score
Double your score with a x2 ability
Each block's score is now multipled twice. Achieve a higher score quicker.
Touch of Midas
Be greedy, take them all!
Every block is golden. Each block is also doubled! This ability is rarer.
